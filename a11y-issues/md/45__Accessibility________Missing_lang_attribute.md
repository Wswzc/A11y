# [Accessibility][如何使用] Missing lang attribute

**页面**: 如何使用

**优先级**: Serious

**标签**: lang

**问题描述**:

Page: 如何使用
Document <html> element has no lang attribute (lang is empty).
Recommendation: add <html lang="zh-CN"> or appropriate language.

**复现步骤**:

检查页面源代码，确保 <html> 有正确的 lang 属性。
